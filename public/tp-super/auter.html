Auteur

Ce site a été développé par Brillon Baptiste, un passionné de superhéros et de développement web. L'objectif de ce projet est de fournir une plateforme interactive pour explorer les superhéros et leurs pouvoirs.

Technologies utilisées :

  • HTML et CSS pour la structure et le style
  • Bootstrap pour la mise en page réactive
  • Vue.js pour la gestion de l'interface utilisateur
